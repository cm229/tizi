*Github梯子*，也称为*翻墙工具*，是指能够帮助用户绕过网络限制，以便顺利访问*Github*及其他被屏蔽网站的工具。常见的*Github梯子*有：

* VPN（虚拟专用网络）

## 如何选择适合的Github梯子？
